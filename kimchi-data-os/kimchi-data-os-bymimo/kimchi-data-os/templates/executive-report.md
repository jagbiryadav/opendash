## Report Quality Checklist

- [ ] Executive summary answers the question in first paragraph
- [ ] Key findings are quantified with business impact
- [ ] Recommendations are prioritized by impact/effort
- [ ] Visualizations support key messages
- [ ] Confidence levels are provided for all findings
- [ ] Limitations and caveats are clearly stated
- [ ] Language is appropriate for audience
- [ ] All claims are supported by evidence
- [ ] Technical details are in appendix, not main text
- [ ] Report is reviewed for accuracy and clarity
