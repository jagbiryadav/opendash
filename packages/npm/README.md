# OpenDash CLI

AI-powered coding agent for the terminal.

## Install

```bash
npm install -g @opendash-ai/cli
```

## Usage

```bash
opendash
```

Or use the shorter alias:

```bash
analyst
```

## Platform Support

| Platform | Architecture |
|----------|-------------|
| macOS | ARM64 (Apple Silicon) |
| macOS | x64 (Intel) |
| Linux | x64 (glibc) |
| Linux | x64 (musl/Alpine) |
| Linux | ARM64 (glibc) |
| Linux | ARM64 (musl/Alpine) |
| Windows | x64 |
| Windows | ARM64 |
