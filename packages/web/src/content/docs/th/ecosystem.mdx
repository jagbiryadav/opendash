ชุดโครงการชุมชนที่สร้างขึ้นบน OpenDash

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OpenDash ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-opendash](https://github.com/awesome-opendash/awesome-opendash) และ [opendash.cafe](https://opendash.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
